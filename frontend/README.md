# Frontend eBA

Este frontend corresponde a la aplicación mobile de eBA. Está desarrollado con React Native, Expo y Expo Router.

## Tecnologías utilizadas

- React Native
- Expo
- Expo Router
- TypeScript
- Lucide React Native

## Requisitos previos

Antes de ejecutar el frontend, tener instalado:

- Node.js LTS
- npm
- Git
- Visual Studio Code
- Expo Go en el celular, opcional

Para verificar Node y npm:

```bash
node -v
npm -v
